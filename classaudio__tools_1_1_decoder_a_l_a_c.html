Detailed Description

ALAC (Apple Lossless Audio Codec) decoder. Please note that this codec usually needs a container (e.g. M4A) to provide the relevant codec information. This class depends on https://github.com/pschatzmann/codec-alac.

Author
Phil Schatzmann
