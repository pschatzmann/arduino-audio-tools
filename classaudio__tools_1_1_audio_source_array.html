Detailed Description

template<typename FileType>
class audio_tools::AudioSourceArray< FileType >

Audio Data Source managing a static array of file names (const char*). Designed for PROGMEM storage on Arduino platforms with static file lists. This class is a template to support multiple SD libraries.
